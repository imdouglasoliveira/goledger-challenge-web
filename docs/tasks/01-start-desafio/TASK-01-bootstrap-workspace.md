# TASK-01 - Bootstrap de workspace

## Objetivo

Criar a fundacao do repositorio para `apps/web` e `apps/api`.

## Por que fazer

- sem estrutura clara, o projeto degrada rapido
- o jr precisa de caminhos previsiveis para backend e frontend
- scripts raiz reduzem erro operacional

## Como fazer

1. Revisar `package.json` raiz e scripts existentes.
2. Criar a estrutura `apps/web` e `apps/api`.
3. Reservar pasta de configuracoes compartilhadas apenas se realmente precisar.
4. Definir scripts raiz para `dev`, `build`, `lint`, `test` e `storybook`.
5. Validar se a estrutura bate com `docs/plans/01-start-desafio.md`.

## Checklist

- `apps/web` criado
- `apps/api` criado
- scripts raiz definidos
- README com comandos basicos atualizado

## Entregaveis

- estrutura inicial criada
- scripts raiz definidos
- README atualizado com comandos basicos
